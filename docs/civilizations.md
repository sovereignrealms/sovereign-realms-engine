# Civilizations — Focus & Roster

**Setting:** ancient world, ~500 BC – 200 AD. Mediterranean + East Asian theatres.

## Primary civilizations (initial roster focus)

### Mediterranean

- **Roman Empire** — most iconic silhouettes, cleanest reference imagery
- **Macedonian / Alexandrian** — Alexander's army, distinctive heavy cavalry
- **Greek / Spartan** — Hoplite warfare aesthetic

### East Asian

- **Qin (Warring States China)** — Warring States → Qin unification (~475–206 BC). Iconic units: Qin spearman, Qin crossbowman (the Qin army's signature weapon system), heavy infantry in lamellar armor, light cavalry, war chariot. Strong visual differentiation: black + bronze color palette, overlapping lamellar plates, longer polearms (*ji* halberds), recurve bows. Reference: Terracotta Army (PD photographs) + *Kingdom* manga.

## Adjacent civilizations (broader roster, future)

### Mediterranean / Near East
- **Persian** (Achaemenid Immortals, scaled armor, kandys robes)
- **Carthaginian** (Sacred Band, Numidian cavalry, war elephants)
- **Egyptian** (Late Period / Ptolemaic — chariotry, archers)

### East Asian
- **Zhao** (Warring States rival to Qin — famous cavalry, northern frontier)
- **Chu** (southern Warring States — distinctive shamanic visual culture)
- **Han Dynasty** (~206 BC – 220 AD — successor to Qin, iron-age refinement)

## Cross-cultural roster size

Six primary civs × ~6–8 unit types each ≈ 36–48 base units to design. Adjacent civs add another ~30+ units when scoped in. This is large; staged delivery is essential — see asset pipeline doc.
