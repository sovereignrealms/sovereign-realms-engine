# Sovereign Realms Worlds

This folder is for world and game packs that run on the Sovereign Realms engine
fork.

Each pack must live in its own folder:

```text
games/<pack_id>/
```

Each pack must include:

```text
LICENSE
README.md
world.json
```

World packs may use their own license, including MIT, CC-BY, CC0, or another
license selected by the contributor. That license applies to the pack's
original scripts, maps, data, and assets. It does not relicense root engine
code.

See:

- `docs/modding/licensing_worlds.md`
- `docs/modding/world_pack_format.md`
