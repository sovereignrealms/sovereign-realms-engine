# Sovereign Realms Engine Notice

This repository is a Permafrost Engine-derived work for the Sovereign Realms
Metal-native RTS engine effort.

The intended public fork target for this line of work is
`sovereignrealms/sovereign-realms-engine`.

Permafrost Engine was created by Eduard Permyakov and is distributed under the
GNU GPLv3 with the special linking exception preserved in `LICENSE.txt` and in
source file headers.

Sovereign Realms modifications are intended to keep the engine open, preserve
the upstream license notices, and separate original game data and assets from
the engine code wherever practical. Alternate licensing, including Apache 2.0
or dual licensing, requires explicit permission from the upstream copyright
holder before it can be applied to Permafrost-derived engine code.

No Microsoft, Ensemble Studios, or Age of Empires assets are part of this
repository. Age of Empires II: Definitive Edition is used only as a gameplay
and readability benchmark for RTS engine planning.
